This is necessary so that the plugin receives the correct <strong>authorization token</strong> from Google.<br /><br />Finally, this version introduces improvements in <strong>Google Drive</strong>, <strong>Microsoft OneDrive</strong>, <strong>Dropbox</strong> and <strong>Amazon S3</strong> functionalities so that the admin is <strong>notified</strong> if there is something wrong with their activation.<br /><br />
